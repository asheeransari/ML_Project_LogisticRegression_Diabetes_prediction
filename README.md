## ML_Project_LogisticRegression_Diabetes_prediction

## Project Info
This Machine Learning Project Predict the Diabetes Problem with the help of other features and Classify the output whether the patient diabetic or not. This is the simple Machine Learning Project that use Logistic Regression Algorithm to predict the output.
